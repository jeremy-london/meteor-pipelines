&nbsp;

Docker image for BitBucket Pipelines CI with Meteor & all required stuff for E2E and other testing

Includes (among other):

- Meteor (latest official release at the time of the build)
- Node 8
- npm 5
- Java 8
- PhantomJS 2
- SonarQube Scanner 3
